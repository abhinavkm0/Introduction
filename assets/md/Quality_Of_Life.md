# QoL Addons

The following are entirely optional

## ZSH

Install Zsh

```
sudo apt-get install zsh
```

Install OhMyZsh

```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

You can configure your .zshrc as you wish

```
nano ~/.zshrc
```
